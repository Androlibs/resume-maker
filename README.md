## Resume maker

Python 3+

usage:
```
cd app

make.py <command>
```

#### Commands:
  create - Create a resume

  list - List templates


#### how use create?
```
cd app

make.py create <lang> <template> <data_json> <output_path>
```

#### langs
pt = portuguese

en = english

es = spanish
